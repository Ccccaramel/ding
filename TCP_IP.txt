TCP/IP(Transmission Control Protocol/Internet Protocol) 
  指传输控制协议/网际协议,意味着 TCP 和 IP 在一起协同工作
  TCP 负责应用软件和网络软件之间的通信
  IP 负责计算机之间地通信
  TCP 负责将数据分割并装入 IP 包并在它们到达时重新组装
  IP 负责将包发送到接收者
  因特网/电子邮件的通信协议
  它定义了电子设备如何连入因特网以及数据如何在它们之间传输的标准

计算机通信协议(Computer Communication Protocol)
  指计算机必须遵守以便彼此通信的规则描述

包含用于处理数据通信的协议
  TCP(传输控制协议):应用程序之间通信
  UDP(用户数据报协议):应用程序之间的简单通信
  IP(网际协议):计算机之间的通信
  ICMP(因特网消息控制协议):针对错误和状态
  DHCP(动态主机配置协议):针对动态寻址

TCP
  使用固定的连接
  将两个应用程序之间建立一个全双工(full-duplex)的通信,直至一方或双方关闭

IP
  无连接
  用于计算机之间的通信
  它不会占用两个正在通信的计算机之间的通信线路

IP 路由器
  当一个 IP 包从一台计算机被发送,它会到达一个 IP 路由器
  IP 路由器负责将这个包路由至它的目的地,直接地或通过其它路由器
  在一个相同的通信中,一个包所经由的路径可能会和其它的包不同
  而路由器负责根据通信量,网络中的错误或其它参数进行正确地寻址
